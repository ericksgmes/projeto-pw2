Restaurante API

Este webservice gerencia o sistema de um restaurante, permitindo o controle de funcionários, mesas, produtos e pagamentos. É possível cadastrar, consultar, atualizar e excluir informações relacionadas a esses itens.

Tabela de Rotas de Funcionários

Método Exemplo Descrição Resultado
GET GET /funcionarios Obtém todos os funcionários 
[
    {
        "id_funcionario": 1,
        "nome": "João Silva",
        "username": "joao.silva"
    },
    {
        "id_funcionario": 2,
        "nome": "Maria Oliveira",
        "username": "maria.oliveira"
    }
]
GET GET /funcionarios/1 Obtém um funcionário específico 
{
    "id_funcionario": 1,
    "nome": "João Silva",
    "username": "joao.silva"
}
POST POST /funcionarios Cadastra um novo funcionário 
{
    "status": "success",
    "data": 3,
    "links": [
        {"rel": "self", "href": "/funcionarios/3"},
        {"rel": "update", "href": "/funcionarios/3"},
        {"rel": "delete", "href": "/funcionarios/3"}
    ]
}
PUT PUT /funcionarios/1 Atualiza os dados de um funcionário 
{
    "status": "success",
    "data": {
        "id_funcionario": 1,
        "nome": "João Silva",
        "username": "joao.silva"
    },
    "links": [
        {"rel": "self", "href": "/funcionarios/1"},
        {"rel": "delete", "href": "/funcionarios/1"}
    ]
}
